Auto Scratch Desktop Mirror 是一个 Scratch 桌面版的开源下载镜像项目。我们通过 Github Workflow 自动下载 Scratch 官网的 Scratch 桌面版程序，并透过 CDN 或 Github Pages 提供，以方便中国大陆地区的用户下载。

项目开源于 GitHub，本版本由 waterblock79 开发。
